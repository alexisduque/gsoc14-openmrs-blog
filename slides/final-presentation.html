ATLAS SERVER

https://atlas.openmrs.org

  • Most  of the work  accomplished during 1st part of the summer  - current release is 2.0.2.

  • Focus on interaction with the module :
      • Authentication using UUID token.
      • URL Routing  /module with POST/GET verbs .
      • Cross-origin PostMessage API,  "Module only" JS files.

  • Extra feature : RSS feed, Sharing link .

ATLAS MODULE



  • Make it working as an OpenMRS 2.0 Application with  App & UI Frameworks.
  • Compatible from 1.8.x to 2.0+.

  • Module become just way to automate markers update on OpenMRS Atlas.

  • Collects implementation information from administrators and send it every week  to atlas server.

  


FURTHER WORK


  • Collect server implementation configuration and display them to the Atlas.

  • Integrate Atlas during OpenMRS installation process.

  • Atlas internationalization.

  • Atlas authentication using ID API and OAuth
  • Interaction with the new ID data model
